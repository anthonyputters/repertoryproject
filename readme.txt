======= Le Projet CinePose =======

Repertory Project est une application web de gestion 
de contact de type CRUD. Elle a été développé en Java 
avec le Framework Spring MVC. Les pages web sont au 
format .jsp. Elles se trouvent dans le repertoire : 
src/main/webapp/WEB-INF/views/ 
Les fichiers contenant le css se trouvent dans le 
repertoire : src/main/webapp/resources/css/ 
Les fichiers contenant le javascript se trouvent dans 
le repertoire : str/main/webapp/resources/js/

Fonctionnalités :
- Ajout de contact et d'adresses associées
- Affichage des contacts enregistrés
- Affichage de chaque contact en détail
- Modification des contacts
- Suppression de contacts et d'adresses
- Recherche parmi les contacts

Ce projet à été développer dans le cadre d'un cours 
d'Intarface Homme-Machine de l'école d'ingénieur
